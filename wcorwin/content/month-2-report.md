## What still needs to happen

### Content deployment

The 7 pages are written and optimized but need to be published on wcorwin.com. There are three options:

1. iHouseWeb blog module, if available. Publish as blog posts. This requires activating the blog module and confirming URL structure.
2. Standalone landing pages built as independent pages within iHouseWeb using the page editor.
3. External content pages hosted on a subdomain or integrated platform. Less ideal for SEO because domain authority does not transfer as effectively.

Option 2 (standalone pages within iHouseWeb) is the recommended path. It keeps all content on the main domain and gives full control over URL slugs, meta tags, and schema.

### Internal link implementation

The linking map is ready. Links need to be added to existing pages via the iHouseWeb page editor and built into new page templates.

### FAQPage schema deployment

Each new page needs page-specific FAQ schema injected. This can be done through the iHouseWeb "Extra Code for Head" field, either per-page if available or assembled for global injection with page conditionals.

## Month 3 preview: authority building

Content is the foundation. Month 3 shifts to authority, the third pillar of the 3-month roadmap.

Citation building: New Braunfels Chamber of Commerce listing, Comal County directories, Texas realtor directories, and niche platforms like HomeLight and FastExpert. Target is 10+ new citations with consistent NAP.

GBP optimization: updated business description with keywords, weekly GBP posts (neighborhood spotlights and market updates), photo management (GBP photos are already processed and uploaded), and Q&A management with 5 to 10 seeded questions and optimized answers.

Backlink outreach: local veteran organizations (VFW, American Legion) for partnership or sponsorship links, community event sponsorships with backlinks, and guest content opportunities on local news and blog sites.

FAQ schema expansion: adding FAQPage schema to existing service and agent pages, and monitoring FAQ rich results in GSC.

90-day performance review: full GSC analysis (clicks, impressions, CTR, position) compared against the March baseline (246 clicks, 25.6K impressions, 1% CTR, avg position 11.5), a ranking report for all target keywords, and an adjusted strategy for months 4 and beyond.
