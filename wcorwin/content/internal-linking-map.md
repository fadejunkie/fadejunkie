## Strategy

The core principle: every new content page should receive links from existing high-authority pages, and should link to other relevant new and existing pages. This creates a **hub-and-spoke** architecture where:

- **Hub pages** = Homepage, About Us, Our Agents (high authority, existing)
- **Service spokes** = Buyer Rebate, VA Home Loans, First-Time Buyer (new landing pages)
- **Location spokes** = Canyon Lake, Gruene, Spring Branch, Seguin (new neighborhood guides)
- **IDX spokes** = Existing community listing pages (homes for sale in [city])

Each spoke links back to the hub and cross-links to related spokes. This distributes PageRank efficiently and creates topical clusters that signal expertise to Google.

## Notes for Funkie Implementation

When these pages are built on the wcorwin.com platform (either via iHouseWeb or standalone landing pages):

1. **Existing pages (iHouseWeb):** Links need to be added through the page editor ‚Äî find the relevant text section and wrap keywords in `<a href="/target-page">anchor text</a>`
2. **New pages (standalone):** Links should be built into the page HTML/template during development
3. **Navigation:** Consider adding a "Resources" or "Guides" dropdown in the main nav that links to the three landing pages and four neighborhood guides
4. **Footer:** Add a "Neighborhood Guides" section in the footer linking to all four guides
5. **Sidebar/CTA:** If building a content template, include a persistent sidebar CTA that links to the buyer rebate page
