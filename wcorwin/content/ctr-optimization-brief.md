## Implementation Notes

### For New Pages (items 1–7)
These pages will be built outside of iHouseWeb (as standalone landing pages or via a blog/content module). Title tags and meta descriptions should be hardcoded in the `<head>` of each page.

### For Existing Pages (items 8–12)
Apply via iHouseWeb admin → Advanced Page Settings → SEO fields (same process as Month 1).

### Schema Additions for New Pages
Each new page should include page-specific FAQ schema (`FAQPage` JSON-LD) in addition to the existing global `RealEstateAgent` schema. FAQ schema template:

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Question text from FAQ section]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer text from FAQ section]"
      }
    }
  ]
}
```

Pull questions directly from each page's FAQ section.
