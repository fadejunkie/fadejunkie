How Pages Connect
Every navigation path, CTA, and cross-link between pages on fadejunkie.com.

Landing Page CTAs

/ → /signin?mode=signup "Join free" and "Start studying" buttons — all auth-gated CTAs
/ → /directory "Browse directory" — only CTA that doesn't require auth
/ → /signin "Log in" nav link

Post-Auth Redirect

/signin → /home Default redirect after successful login or signup

Dashboard Navigation (Sidebar)

sidebar → /home Home — community feed
sidebar → /profile My Profile — barber identity editor
sidebar → /website Website — shop site builder
sidebar → /tools Tools — study hub & utilities
sidebar → /resources Resources — curated affiliate directory
sidebar → /directory Directory — school/shop/supply map

Cross-Page Links

/home → /barber/[slug] Post author @handle links to their public profile
/profile → /barber/[slug] "View your live page" card shows public profile URL
/website → /shop/[userId] "View your live page" card shows public shop URL
/tools → /tools/flashcards Flashcards tool card
/tools → /tools/practice-test Practice test tool card
/tools → /tools/exam-guide Exam guide tool card
/tools → /website Website builder tool card

External Exits

/barber/[slug] → external Booking link, phone call, Instagram profile
/resources → external Affiliate links to vendor/tool websites
/directory → external Location website, phone number from modal
