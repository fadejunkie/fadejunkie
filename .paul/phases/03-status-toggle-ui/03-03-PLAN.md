<objective>
## Goal
Add a `getMyArchivedStatuses` query that returns the current user's expired/deactivated statuses, and task Lobe with building the archive history UI below the active toggles on the /status page.

## Purpose
Users should see their status history -- what they previously broadcast and when it expired. This closes the status lifecycle loop: activate, expire, review. Without it, archived statuses are invisible dead data.

## Output
- `getMyArchivedStatuses` query added to `app/convex/statuses.ts`
- Lobe task brief at `lobe/inbox/status-history-view.md` for the archive view UI
</objective>
