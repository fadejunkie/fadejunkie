<objective>
## Goal
Add a `userPaths` table to Convex that tracks which path(s) each user has selected, and create mutations/queries for path management. Users can have multiple paths (a barber who also owns a shop).

## Purpose
The status toggle system needs to know what path a user is on to show them the right toggles. This plan creates that data layer. Phase 2's later plans will add the UI for path selection.

## Output
- `userPaths` table added to `app/convex/schema.ts`
- `app/convex/userPaths.ts` — mutations and queries for path management
</objective>

    Add a `userPaths` table to the schema, placed right after the `statuses` table in the Status ecosystem section.

    Fields:
    - userId: v.id("users")
    - path: v.string() — the user path (barber, student, shop, etc.)
    - isPrimary: v.boolean() — whether this is the user's primary path
    - createdAt: v.number()

    Indexes:
    - by_userId: ["userId"] — get all paths for a user
    - by_path: ["path"] — get all users on a specific path

<task type="auto">
  <name>Task 2: Create userPaths.ts with mutations and queries</name>
  <files>app/convex/userPaths.ts</files>
  <action>
    Create `app/convex/userPaths.ts` with:

    **Mutations:**

    1. **selectPath** mutation:
       - Args: path (v.string())
       - Auth: require authenticated user
       - Validate: path is a valid UserPath (check against USER_PATHS)
       - Check: user doesn't already have this path (query by_userId_path)
       - If duplicate: throw ConvexError("Path already selected")
       - Determine isPrimary: true if user has no existing paths, false otherwise
       - Insert: { userId, path, isPrimary, createdAt: Date.now() }

    2. **removePath** mutation:
       - Args: path (v.string())
       - Auth: require authenticated user
       - Find the userPaths record for userId + path
       - If not found: throw ConvexError("Path not found")
       - If isPrimary and user has other paths: promote the oldest remaining path to primary
       - Delete the record

    3. **setPrimaryPath** mutation:
       - Args: path (v.string())
       - Auth: require authenticated user
       - Find all paths for this user
       - Set the specified path to isPrimary=true
       - Set all other paths to isPrimary=false
       - If path not found: throw ConvexError("Path not found")

    **Queries:**

    4. **getMyPaths** query:
       - Auth: require authenticated user
       - Query by_userId for current user
       - Sort: isPrimary first, then by createdAt ascending
       - Return all paths

    5. **getUserPaths** query:
       - Args: userId (v.id("users"))
       - No auth required (public)
       - Query by_userId
       - Sort: isPrimary first
       - Return paths

    Use same auth pattern as barbers.ts (getAuthUserId).
    Use ConvexError for user-facing errors.
    Import USER_PATHS from statusConfig for validation.
  </action>
  <verify>Run `npx convex dev --once` and `npx tsc --noEmit`</verify>
  <done>AC-2, AC-3, AC-4 satisfied: path selection, primary switching, and queries work</done>
</task>
