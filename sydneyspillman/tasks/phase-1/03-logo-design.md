# Logo Design — Sydney Spillman & Associates

**Primary agent:** MANUAL (escalation)
**Support:** Ink (mockup copy, presentation text)

## Why Manual

Logo design requires iterative visual creation that agents cannot fully execute — concept generation, vector refinement, client presentation with physical mockups. This milestone is a **manual checkpoint**.

## When This Triggers

After `1-MOOD + DIRECTION-2` (client approval on direction) is marked complete.

## Escalation

Dispatch should write to `dispatch/escalations/sydney-logo-design.md`:

```
## Logo Design — Manual Checkpoint

The mood + direction milestone is complete. Client approved direction: [A or B].
See `sydneyspillman/content/02-direction-options.md`.

### What Anthony Needs to Do
1. Generate 3 distinct logo concepts (AI-assisted + manual vector refinement)
2. Present concepts with mockups — business cards, signage, social, yard signs
3. Revision round 1 on selected direction
4. Revision round 2 — final polish and lockup variations
5. Export final files: PNG (transparent), SVG (vector), PDF (print-ready)
6. Create icon-only and wordmark-only variants

### Task Keys to Mark Complete
- 1-LOGO DESIGN-0 through 1-LOGO DESIGN-5

### Where to Put Assets
- Logo files → `sydneyspillman/content/brand/logo/`
```
